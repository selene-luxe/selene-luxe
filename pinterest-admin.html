How to Post in 5 Steps
1. Click Create Pin on Pinterest — opens Pinterest pre-filled with the image, link & description.   2. Select a board, then click Edit (pencil icon) to open the full pin editor.   3. In the Title field: the title is already shown as the first line of the description — select it, cut it, paste it into the Title box.   4. Or click Copy Title first & paste it (Ctrl+V).   5. Add the overlay text, tag suggested topics, then hit Publish.
